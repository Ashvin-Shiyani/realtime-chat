# Real-Time Chat Application

Hello, I built this real-time chat web application using Node.js, Express and Socket.io.
I started this project to learn how real-time communication works on the web
and to get hands on experience with Node.js and WebSockets.

## Features I Built
- Users can enter a username and join the chat
- Messages are sent and received in real time
- You can see how many users are online at the same time
- Notifications when someone joins or leaves the chat
- Typing indicator that shows when someone is typing

## Tech Stack I Used
- Node.js
- Express
- Socket.io
- HTML, CSS, JavaScript

## How to Run
1. Run npm install
2. Run npm start
3. Open http://localhost:3000 in your browser

## How it Works
I used server.js to create the server and handle all real time
communication between users using Socket.io. The index.html is
the chat interface that connects to the server and lets users
send and receive messages instantly.
